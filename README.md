# Iris Flower Classification

This project uses Machine Learning to classify iris flowers into:
- Setosa
- Versicolor
- Virginica

## Dataset
The Iris dataset is a built-in dataset from scikit-learn.

## Algorithm Used
- Logistic Regression

## Steps Involved
1. Load Iris dataset
2. Split data into training and testing
3. Train ML model
4. Predict flower type
5. Evaluate accuracy

## How to Run
```bash
pip install -r requirements.txt
python iris_classification.py
